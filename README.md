**Open Atlas** is a community-driven project to create a powerful, open-source, and cross-platform alternative to ChatGPT's proprietary Atlas browser. We believe in democratizing access to AI-powered browsing for everyone, regardless of their operating system. Our mission is to provide a transparent, secure, and feature-rich browsing experience that harnesses the power of artificial intelligence for all.

This project is for you if you're looking for an open-source ChatGPT Atlas alternative that runs on Windows, macOS, and Linux.
